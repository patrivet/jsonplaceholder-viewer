JSON Placeholder viewer
=====
1. Prerequisites
	a. Install nvm, if not already installed.
		curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
	b. Install node, if not already installed.
		nvm install node

2.How to Run from the Command Line

	Make sure you are inside the 'jsonplaceholder-viewer' folder.

	a. First install the npm dependencies:

			npm install

		Run using npm:

			npm start
			
			Go to http://localhost:3000/ to view..

3. If you want to run tests:

	npm test


4. Live app:
To see the app running - visit the hosted site (https://json-placeholder-viewer.netlify.app/) on Netlify.
